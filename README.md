
# 🪔 Diwali Sales Data Analysis (EDA Project)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Diwali sales dataset to uncover customer purchasing behavior, sales trends, and business insights.

The goal is to analyze customer demographics and buying patterns to help businesses make **data-driven marketing and sales decisions**.

---

## 📂 Dataset Used

* **File Name:** `Diwali Sales Data.csv`
* Contains customer-level transactional data including:

  * User ID
  * Gender
  * Age Group
  * State
  * Occupation
  * Product Category
  * Orders
  * Amount Spent

---

## 🛠️ Technologies & Libraries

* Python 🐍
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔎 Steps Performed

1. **Data Loading**

   * Imported CSV file using Pandas.

2. **Data Cleaning**

   * Removed unnecessary columns.
   * Handled missing values.
   * Converted data types where required.

3. **Data Exploration**

   * Checked shape, info, and statistical summary.
   * Identified null values and duplicates.

4. **Data Visualization**

   * Gender-wise purchasing analysis
   * Age group spending trends
   * State-wise total sales
   * Occupation-based purchasing power
   * Product category performance

---

## 📊 Key Insights

* 👩 Female customers contributed more to total sales compared to male customers.
* 🎯 Age group **26–35 years** showed the highest purchasing power.
* 🏙️ Certain states contributed significantly higher revenue.
* 💼 Working professionals and IT sector customers showed strong spending trends.
* 🛍️ Specific product categories performed better during Diwali season.
